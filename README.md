# 한화 ATLAS — Agentic Investment Desk OS

> 바이브코딩 경진대회 · 주식운용 아이디어 부문 제출본
> AI 에이전트가 시장을 읽고 실제로 토론해 투자 아이디어를 발굴·심의하는 주식운용 데스크 OS입니다.

한화손해보험의 주식 포트폴리오 운용 업무 — 시장 모니터링, 손익·위험 점검, 시황 브리핑, 아이디어 발굴·백테스트, 멀티에이전트 투자위원회 — 를 하나의 화면(5개 탭)에서 연결합니다.

---

## 심사 안내 — 오프라인본으로 평가해 주세요

라이브(서버) 버전은 실시간 데이터·LLM 구동에 **API 키가 필요**하며, 보안상 키는 저장소에 포함하지 않았습니다.

**키 없이 바로 확인하시려면 오프라인본 `index.html`을 더블클릭**해 주세요.
최근 실데이터(시세·뉴스·섹터·손익·위원회 결과)가 HTML에 내장되어 있어 **백엔드·설치·인터넷 없이 전 탭·팝업·차트가 그대로 동작**합니다. 우상단 "제출용 · 오프라인 모드" 배지로 정적 데모임을 표시합니다.

### 2분 추천 동선
1. `index.html` 더블클릭 → 설치 없이 브라우저에서 열림
2. 상단 라이브 테이프에서 시세 흐름 확인
3. **손익현황** 탭 → 보유종목 행 클릭 → 우측 상세 Drawer
4. Drawer의 **"AI 위원회 소집"** → 위원회 탭으로 자동 이동 + 종목 자동 세팅(원클릭)
5. **AI 아이디어랩** 탭 → "최근 결과" → 11개 역할 에이전트의 5단계 토론 회의록·후보 종목 확인
6. **시황에이전트** 탭 → 장전/장중/마감 브리핑 확인

## 주요 기능

### 1) 시장현황
- KR 준실시간 시세 테이프(네이버 금융 폴링 + pykrx 폴백)와 KOSPI·KOSDAQ·USD/KRW·VIX·WTI·금 KPI
- 섹터 등락, 멀티에셋 시세 테이블, 지수/종목 캔들 차트
- 시세 기준일과 수집 시각을 분리 표기해 데이터 신뢰도 투명화

### 2) 손익현황
- 인포맥스 마스터 엑셀을 직독해 보유종목 평가·손익 집계
- 손익 시계열 곡선(포트 시작=100 리베이스 + 벤치마크 오버레이)
- vs BM 위험지표: 연율수익·변동성·MDD·베타·추적오차·정보비율·초과수익
- 보유종목 행 클릭 → 상세 Drawer → 원클릭으로 AI 위원회 소집

### 3) 시황에이전트
- 장전/장중/장마감 슬롯별 LLM 시황 리포트(이슈·강세/약세 시나리오·전략·뉴스 흐름 등 9개 섹션)
- 상대강도(RS) 4분면, ADR 추이, 등락 종목·섹터 차트, 헤드라인
- PNG 리포트 미리보기·다운로드, 발송 이력·다음 자동생성 카운트다운

### 4) AI 아이디어랩
- RAG 근거접지 투자 아이디어: 시세·펀더멘털·수급·공매도·뉴스를 수집해 LLM이 thesis·목표가·핵심동인·리스크를 근거(출처) 카드와 함께 생성
- 기관급 전략 백테스트 5종(MA 크로스·듀얼모멘텀·볼린저·52주 돌파·목표변동성)
- CAGR·MDD·샤프·Sortino·Calmar·승률 + 자본곡선·매매 마커·월별 히트맵, 거래비용·체결규칙·면책 가정 함께 노출

### 5) AI 투자위원회 (멀티에이전트 심의)
- **실제로 토론하는** 톱다운 발굴·심의 엔진(연출이 아닌 실 토론):
  1. **사전조사·발굴** — Macro PM + 스카우트가 실시간 섹터·뉴스·6팩터로 후보풀 동적 발굴
  2. **섹터 라운드테이블** — Bull ↔ Bear 멀티라운드 토론 → 리서치 매니저 판정
  3. **종목 상정** — 스톡피커가 승리 레인에서 thesis 근거로 지명
  4. **리스크 사전심의** — 공격·중립·보수 3-way 토론 → 리스크 매니저 판정
  5. **PM 의장 최종선정** — 회의록 종합 → 숏리스트 랭킹 + 채택 근거·체크리스트
- 11개 역할 에이전트, 실행당 약 14~18회 LLM 호출, 실시간 발언 스트리밍, 전 단계 규칙 기반 폴백 내장

## 기대 효과

- **의사결정 속도** — 시장 모니터링·손익 점검·시황·아이디어·위원회 심의를 한 화면에서 끊김 없이 연결해 분산된 툴 전환 비용을 제거
- **아이디어 발굴의 폭과 깊이** — 단일 모델 추천이 아니라 다(多)역할 에이전트의 상반된 시각(강세/약세, 공격/보수)을 토론으로 충돌시켜 사각지대를 줄임
- **근거 추적성** — 모든 아이디어·시황·심의 결과에 출처 근거를 함께 제시해 운용역이 판단 과정을 검증 가능
- **운용 신뢰성** — LLM/외부 데이터 장애 시 규칙 기반 폴백으로 graceful degrade, 시세 기준일·커버리지·면책 가정을 투명 표기

## 작동법

### A. 오프라인 즉시 실행 (서버 불필요 · 심사 권장)
`index.html` 더블클릭 → 설치·인터넷 없이 전 기능 동작. GitHub Pages를 켜면 동일 파일이 그대로 온라인 데모 URL이 됩니다.

### B. 풀스택 실행 (실시간 데이터 + 실제 LLM 토론)
```powershell
# 백엔드 (FastAPI, Python 3.9)
Set-Location ".../ai-investment-desk-os/backend"
$env:PYTHONPATH = (Get-Location).Path
pip install -r requirements.txt
uvicorn app.main:app --reload --host 127.0.0.1 --port 8000

# 프런트엔드 (React 19 + Vite)
Set-Location ".../ai-investment-desk-os/frontend"
npm install
npm run dev   # http://127.0.0.1:5173
```

키(`MIMO_API_KEY` 등)는 `committee_engine/TradingAgents/.env`에 입력하면 백엔드 기동 시 환경변수로 주입됩니다. 키가 없어도 규칙 기반 폴백으로 동작하며, 키를 넣으면 실시간 시세 + 실제 멀티에이전트 LLM 토론이 활성화됩니다.

### 기술 스택
- 프런트: React 19 + Vite + Tailwind (한화 웜다크 터미널 테마)
- 백엔드: FastAPI (Python 3.9)
- LLM: Xiaomi MiMo V2.5 (`MIMO_API_KEY`)
- 데이터: 네이버 금융 · pykrx · yfinance · Google News RSS · 인포맥스 마스터 엑셀 · DART(선택) · ECOS(선택)
